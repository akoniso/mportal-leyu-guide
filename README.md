# mportal-leyu-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常作为信息展示、工具页面或静态内容的入口，不针对任何特定域名或网站。

## 项目简介

`mportal-leyu-guide` 是一个静态资源归档仓库，主要存放结构简单、自包含的 HTML 文件。每个页面独立运行，不依赖后端服务，适合快速部署和浏览。

## 目录说明

```
mportal-leyu-guide/
├── index.html          # 仓库总览或入口页面
├── page-example-1.html # 示例页面一
├── page-example-2.html # 示例页面二
└── assets/             # 公共资源目录（如 CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `index.html`：可选的总览页面，用于列出或导航到其他归档页面。
- 其他 `.html` 文件：每个文件代表一个独立页面，可直接在浏览器中打开访问。
- `assets/`：存放页面共享的样式、脚本、图片等静态资源，可根据需要增减。

## 页面归档说明

- 所有页面均为纯静态 HTML，不包含动态服务端逻辑。
- 页面之间相互独立，无耦合依赖。
